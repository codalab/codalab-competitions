About CodaLab

CodaLab is a unified web-based platform that allows researchers to share and interact with code, data, and experiments, and to participate in online competitions. CodaLab's mission is to facilitate empirical research in fields such as machine learning, medical imaging, and natural language processing.

Develop with Experiments

The experimental platform allows you to upload datasets and algorithms, run algorithms on datasets (either yours or existing ones), and download and visualize the results. The platform keeps track of dependencies, so that experiments are reproducible and easy to share with others.

Compete in Challenges

Competitions provide an opportunity to pit your algorithms against others with the goal of finding the best way to solve a given problem. Problem sets originate from a variety of fields, although in this release the focus is on advancing computer vision through medical image analysis technology. Each competition features a detailed description of the problem, and a set of data against which solutions can be trained and tested.

Your Data in the Cloud

Datasets form the basis for competitions. CodaLab gives you the ability to upload data to Microsoft Azure™ for use in competitions. Datasets can include CT scans, MRIs, and other types of images. Each dataset represents a problem or set of problems to solve. For example, a competition could be created to find the most efficient way of segmenting mitochondria in a scanned cross-section of a human brain.

Glossary

  • Competition - An organized way for users to upload either a program or a bundle with predictions. An evaluation program will automatically score these predictions against the dataset provided by the competition organizer.
  • Bundle - An immutable directory which could contain the code or data. A Bundle can depend on other Bundles.
  • Run - A type of Bundle which is equipped with a command-line string, which specifies how to execute the code in the Bundle. Execution produces an output directory and various statistics.
  • User - Someone who creates programs, datasets, experiments, and participates in competitions.
