basic_ml

By: Percy Liang

This worksheet performs the basic machine learning pipeline for evaluating a single learning algorithm on a single dataset. We first split the dataset into two parts, one for training and the other for testing. We use the learning algorithm to learn a model on the training data, and then use that model to make predictions on the test data. A separate evaluation module is used to report the error (number of misclassified examples).
